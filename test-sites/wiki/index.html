Rust (programming language)

Rust is a multi-paradigm, general-purpose programming language that emphasizes performance, type safety, and concurrency. It enforces memory safety without a garbage collector.

Overview

Rust was originally designed by Graydon Hoare at Mozilla Research in 2010. It has been consistently voted the most admired programming language in the Stack Overflow Developer Survey every year since 2016.

Key Features

  • Ownership system - zero-cost abstractions for memory safety
  • Trait-based generics - compile-time polymorphism
  • Async/await - efficient asynchronous programming
