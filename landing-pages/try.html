Enter any URL and a question. Slaash will distill the page to just the answer.

How Slaash works in production: Question → LLM expands query → Slaash extracts → Answer → LLM sends feedback → Slaash learns. This playground skips the feedback step, so results improve with use in the real pipeline but not here. First queries on a new URL are cold — repeated queries with feedback get significantly better.
