# AetherAgent – Utvecklingsplan 2026-03-21
**Status:** Aktiv
**Datum:** 2026-03-21

---

## Sammanfattning

AetherAgent har nått Fas 16 – alla kärnfaser är klara. Projektet har gått från ren HTML-parser till en fullständig LLM-native browsermotor med:

- Semantisk DOM-parsing med trust shield
- JS-sandbox (Boa) med XHR-interception
- Vision (YOLOv8/ONNX) med tierad rendering
- Sessionshantering, workflow-orkestration
- Adaptiv streaming med LLM-styrda direktiv
- MCP-server, HTTP API, WASM API
- Causal graph, multimodal grounding, collab diffing
- DDG-sökintegration (`search.rs`)

---

## Nästa faser – Roadmap

### Fas 17: Boa JS Utredning & Förbättring
**Prioritet:** Hög
**Branch:** `claude/investigate-boa-js-KLiO4`

Utred nuvarande Boa JS-sandbox begränsningar och förbättra:

1. **Analys av js_eval.rs & js_bridge.rs**
   - Kartlägg vilka JS-mönster som stöds idag
   - Identifiera vanliga patterns som misslyckas
   - Dokumentera Boa-versionens begränsningar

2. **DOM-stub-förbättringar**
   - Utöka `document.getElementById` / `querySelector` stöd
   - Lägg till `classList`, `style`, `dataset` stubs
   - Hantera `addEventListener` grundläggande

3. **Testmatris**
   - Skapa testfall för top-20 JS-patterns på riktiga webbsidor
   - Mät success rate före/efter förbättringar

4. **XHR/Fetch-sandbox**
   - Verifiera att `intercept.rs`-integration fungerar korrekt
   - Testa edge cases: CORS, redirects, streaming responses

---

### Fas 18: Produktionshärdning
**Prioritet:** Medel

1. **Prestandaoptimering**
   - Profilera parse-pipeline (HTML → semantic → JSON)
   - Optimera `stream_engine.rs` för sidor med 1000+ noder
   - Minska WASM-binärstorlek (mål: <4MB)

2. **Felhantering & Resiliens**
   - Audit alla `.unwrap()` i non-test-kod
   - Förbättra felmeddelanden i MCP-tools
   - Timeout-hantering i fetch-kedjan

3. **Säkerhet**
   - Penetrationstesta trust shield med nya injektionsmönster
   - SSRF-audit av fetch.rs
   - Rate limiter-stresstester

---

### Fas 19: Utökad Webbförståelse
**Prioritet:** Medel

1. **CSS-medveten parsing**
   - Extrahera computed styles för layout-förståelse
   - Detektera `display:none`, `visibility:hidden` korrekt
   - Förstå flexbox/grid-layout semantiskt

2. **iframe-hantering**
   - Rekursiv parsing av iframe-innehåll
   - Trust-nivå per iframe-origin
   - Sandboxad JS-exekvering per frame

3. **Shadow DOM**
   - Stöd för open shadow roots
   - Semantisk sammanslagning av shadow tree + light DOM

---

### Fas 20: Agent-protokoll & Ekosystem
**Prioritet:** Låg (framtid)

1. **A2A (Agent-to-Agent)**
   - Implementera Google A2A-protokoll
   - Agent card med capabilities
   - Task lifecycle (pending → working → done)

2. **MCP 2.0**
   - Uppgradera till senaste MCP-specifikation
   - Streamable HTTP transport
   - OAuth 2.1-autentisering

3. **Plugin-system**
   - Dynamisk laddning av parsers per domän
   - Custom trust-regler per agent
   - Webhook-integration för events
